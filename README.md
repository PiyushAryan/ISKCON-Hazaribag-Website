# ISKCON Hazaribag

Official website of ISKCON Hazaribag - Sri Sri Radha Madhava.

The website shares information about the temple, festivals, spiritual teachings, yatras, live darshan, contact details, and donation opportunities.

## About

ISKCON Hazaribag is dedicated to sharing the timeless teachings of Bhagavad-gita and Srimad-Bhagavatam, serving the community, and helping people connect with Krishna consciousness through kirtan, prasadam, festivals, and devotional service.

## Website

Visit the official website to:

- Learn about ISKCON Hazaribag
- Explore festivals and spiritual programs
- Watch live darshan
- Read about Srila Prabhupada and Krishna conscious teachings
- Support temple services through donations
- Contact the temple
